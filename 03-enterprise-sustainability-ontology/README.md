# 🌱 Enterprise Sustainability Ontology & Data Stewardship — Nous Logic Inc.

**Module:** IS6072 Data Analytics for Business Enterprise | University College Cork  
**Client:** Nous Logic Inc.  
**Team Size:** 5 members  
**Year:** 2025–2026

---

## 📌 Project Overview

This project was delivered for a **real client — Nous Logic Inc.** — and involved designing a comprehensive **sustainability ontology** and **Data Stewardship & Reporting System (DSRS)** to support ESG reporting and data governance.

---

## 🎯 Objective

- Design a sustainability-focused data ontology for enterprise use
- Build a data stewardship framework with clearly defined governance roles
- Develop dashboards for ESG reporting and stakeholder communication
- Ensure regulatory compliance and data quality across the pipeline

---

## 🔬 Methodology

| Step | Approach |
|---|---|
| Ontology Design | Graph-based data modelling using Neo4j |
| Data Pipeline | Azure Data Factory for integration and transformation |
| Data Governance | Collibra for stewardship, cataloguing, and compliance |
| Visualisation | Power BI and Tableau dashboards for ESG reporting |
| Stakeholder Output | Data flow maps, stewardship role definitions, compliance documentation |

---

## 📊 Key Deliverables

- A fully structured **sustainability ontology** mapping entities, relationships, and ESG attributes
- A **Data Stewardship & Reporting System (DSRS)** with defined roles and responsibilities
- **Power BI and Tableau dashboards** for real-time ESG performance monitoring
- **Data flow diagrams** and regulatory compliance documentation

## 🛠️ Tools & Technologies

`Neo4j` `Azure Data Factory` `Collibra` `Power BI` `Tableau` `Data Governance` `ESG Reporting`

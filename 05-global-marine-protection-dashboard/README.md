# 🌊 Global Marine Protection (GMP) Dashboard

**Module:** IS6051 Business Data Analysis & Visualisation | University College Cork  
**Team Project**  
**Year:** 2025–2026

---

## 📌 Project Overview

This project involved building an **interactive data dashboard** to visualise global marine protected area (MPA) coverage and conservation trends. The goal was to present complex environmental data in a clear, stakeholder-ready format to support marine conservation policy and decision-making.

---

## 🎯 Objective

- Analyse global MPA coverage data across regions and time periods
- Clean and transform geospatial and time-series environmental datasets
- Build interactive dashboards to communicate conservation trends clearly
- Design stakeholder-focused visual narratives to support policy decisions

---

## 🔬 Methodology

| Step | Approach |
|---|---|
| Data Collection | Global MPA coverage datasets (geospatial & time-series) |
| Data Cleaning | Handled inconsistencies, missing values, and formatting issues |
| Exploratory Analysis | Identified regional trends and coverage gaps |
| Visualisation | Built interactive dashboards with drill-down capability |
| Stakeholder Output | Policy-ready visual narratives and summary insights |

---

## 📊 Key Outputs

- Interactive dashboards showing global MPA coverage by region and year
- Time-series visualisations tracking conservation progress over time
- Geospatial maps highlighting protected and unprotected marine zones
- Stakeholder-ready summary insights for environmental policy support

---

## 🛠️ Tools & Technologies

`Tableau` `Power BI` `Python` `Excel` `Geospatial Data Analysis` `Time-Series Visualisation`

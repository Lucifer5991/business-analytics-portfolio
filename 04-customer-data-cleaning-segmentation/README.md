# 👥 Customer Data Cleaning & Segmentation

**Module:** IS6061 Python for Business Analytics | University College Cork  
**Year:** 2025–2026

---

## 📌 Project Overview

This project involved cleaning, transforming, and analysing a **real-world customer dataset** using Python. The goal was to prepare the data for meaningful analysis and apply segmentation techniques to uncover actionable business patterns.

---

## 🎯 Objective

- Clean and prepare a messy real-world customer dataset for analysis
- Handle missing values, duplicates, and outliers systematically
- Perform exploratory data analysis (EDA) to surface key trends
- Apply customer segmentation to identify distinct customer groups
- Document the full workflow clearly in a Jupyter Notebook

---

## 🔬 Methodology

| Step | Approach |
|---|---|
| Data Cleaning | Handled missing values, duplicates, and outliers using pandas |
| Data Transformation | Feature engineering and data type standardisation |
| Exploratory Data Analysis | Statistical summaries, distribution plots, correlation analysis |
| Segmentation | Grouped customers based on behavioural and demographic patterns |
| Documentation | End-to-end Jupyter Notebook with clear business annotations |

---

## 📊 Key Outputs

- A fully cleaned and analysis-ready customer dataset
- EDA visualisations revealing key customer behaviour patterns
- Customer segments with actionable business interpretations
- A well-documented Jupyter Notebook for reproducibility

---

## 🛠️ Tools & Technologies

`Python` `pandas` `NumPy` `Jupyter Notebook` `EDA` `Data Cleaning` `Customer Segmentation`

# 💡 LumenEire GlowConnect — Smart Adaptive Lighting Solution for Cork

**Module:** IS6058 Design Thinking & Innovation | University College Cork  
**Team Size:** 5 members  
**Year:** 2025–2026

---

## 📌 Project Overview

LumenEire GlowConnect is a **smart adaptive lighting concept** designed for Cork city. The project applied the **design thinking methodology** to identify urban lighting challenges and prototype a data-driven solution that optimises energy usage and improves city livability.

---

## 🎯 Objective

- Apply design thinking to identify real urban lighting challenges in Cork
- Develop a smart, adaptive lighting solution concept
- Create user personas, journey maps, and interface wireframes
- Design a dashboard prototype to monitor energy usage and performance metrics

---

## 🔬 Methodology

| Step | Approach |
|---|---|
| Empathise | User research and stakeholder interviews |
| Define | Problem statement and user persona development |
| Ideate | Brainstorming and concept generation sessions |
| Prototype | Wireframes and dashboard mockups using Figma and Miro |
| Test | Concept validation and feedback integration |

---

## 📊 Key Deliverables

- User personas and customer journey maps for Cork city residents
- Wireframe prototypes of the GlowConnect interface built in Figma
- A dashboard prototype concept to monitor real-time energy usage and lighting performance
- A full design thinking report documenting the end-to-end innovation process

---

## 🛠️ Tools & Technologies

`Figma` `Miro` `Design Thinking` `User Persona Development` `Journey Mapping` `Dashboard Prototyping`
